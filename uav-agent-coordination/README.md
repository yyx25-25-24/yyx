# 基于AI Agent的无人机多机协同干扰任务调度系统

## 项目简介
针对防空反导场景下多无人机协同烟幕干扰的复杂任务调度问题，设计并实现了一个多智能体协同决策系统。系统将每架无人机抽象为独立的智能体，通过增强遗传算法实现多智能体的任务分配、路径规划和投弹策略优化，在满足物理约束和时序约束的前提下，最大化对来袭导弹的有效遮蔽时间。

## 核心功能
- 支持从单智能体到多智能体的递进式任务场景
- 基于增强遗传算法的多智能体协同优化
- 三维运动学建模与遮蔽效果精确计算
- 多维度可视化展示（3D轨迹、遮蔽时间线、优化收敛过程）
- 结果自动导出为Excel格式

## 技术栈
- **核心框架**：多智能体协同框架
- **优化算法**：增强遗传算法、差分进化算法
- **建模工具**：Python、NumPy、Pandas
- **可视化**：Matplotlib、3D绘图

## 快速开始
```bash
# 安装依赖
pip install numpy scipy pandas matplotlib openpyxl

# 运行单无人机单弹场景
python main.py --scenario 1

# 运行多无人机多导弹场景
python main.py --scenario 5
```

## 项目成果
- 构建了覆盖 5 种不同复杂度作战场景的多智能体调度模型
- 解决了 45 维高维决策空间的优化问题，算法收敛速度提升 30%
- 实现了对 3 枚来袭导弹 21.2 秒的综合有效遮蔽，相比单无人机单弹方案提升 1423%
- 完成了完整的可视化展示，直观呈现多智能体协同效果
